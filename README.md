# ARCLI بُريمج فنون سطر الأوامر باللغة العربية

بٌريمج مبني بلغة بايثون يُنشئ رسومات بسطر الأوامر يدعم العربية فقط.

استخدم كونسول الخاص بكيدي لتحصل على أفضل نتيجه لأنه يدعم العربية والكتابة من اليمين إلى اليسار.

## المميزات

- يدعم 6 خطوط مختلفة (font1, font2, font3, font4, font5, font6).
- تظهر نتيجة الرسومات من اليمين لليسار حتى في الطرفيات التي لا تدعم العربية.
- لا تحتاج إلّا للبٌريمج نفسه وملف الخط النصي.
- يدخل النص من سطر الأوامر أو بشكل تفاعلي.
- فحص للأخطاء والمدخلات غير الصحيحة.

## الخطوط المتاحة

- `font1` - يستخدم رمز `#` 
- `font2` - يستخدم رمز `@`
- `font3` - يستخدم رمز `$`
- `font4` - يستخدم رمز `%`
- `font5` - يستخدم رمز `&`
- `font6` - يستخدم رمز `V`

## التثبيت

حمل الملفات وانتقل إلى المجلد وشغل ملف البايثون.

```bash
git clone https://github.com/alwaqad/arcli
cd arcli
python arcli.py
```

## الاستخدام

### الطريقة التفاعلية:
```bash
python arcli.py
```
سيطلب منك البرنامج اختيار الخط ثم إدخال النص.

### من سطر الأوامر:
```bash
# مع اختيار الخط
python arcli.py font2 "مرحبا بالعالم"
python arcli.py font3 "الوقاد"
python arcli.py font5 "اهلا وسهلا"

# بدون اختيار الخط (يستخدم الخط الافتراضي)
python arcli.py "كيف الحال"
```
