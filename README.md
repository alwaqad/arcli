# ARCLI بُريمج فنون سطر الأوامر باللغة العربية

بٌريمج مبني بلغة بايثون يُنشئ رسومات بسطر الأوامر يدعم العربية والحركات.

استخدم كونسول الخاص بكيدي لتحصل على أفضل نتيجه لأنه يدعم العربية والكتابة من اليمين إلى اليسار.

## المميزات

- يدعم 6 خطوط مختلفة (font1, font2, font3, font4, font5, font6)
- تظهر نتيجة الرسومات من اليمين لليسار حتى في الطرفيات التي لا تدعم العربية
- يدخل النص من سطر الأوامر أو بشكل تفاعلي
- فحص للأخطاء والمدخلات غير الصحيحة
- يعمل كحزمة نظام أو من مجلد محلي
- تلوين النص بألوان متعددة
- استبدال النقاط بمسافات لتحسين المظهر

## الخطوط المتاحة

- `font1` - يستخدم رمز `#` 
- `font2` - يستخدم رمز `@`
- `font3` - يستخدم رمز `$`
- `font4` - يستخدم رمز `%`
- `font5` - يستخدم رمز `&`
- `font6` - يستخدم رمز `V`

## الألوان المتاحة

- `white` - أبيض
- `grey` - رمادي
- `red` - أحمر
- `green` - أخضر
- `yellow` - أصفر
- `blue` - أزرق
- `pink` - وردي
- `magenta` - أرجواني
- `purple` - بنفسجي
- `cyan` - تركواز

## التثبيت

### التثبيت كملف تنفيذي

يمكنك تنزيل ملف تنفيذي من [صفحة الإصدارات](https://github.com/3ammoori/arcli/releases) أو [اضغط هنا](https://github.com/3ammoori/arcli/releases/download/arcli/arcli-linux-x86_64.zip) لتنزيل آخر إصدار مباشرة.

### التثبيت كحزمة نظام

```bash
curl -sSL https://raw.githubusercontent.com/alwaqad/arcli/main/install-arcli.sh | sudo bash
```

### تثبيت خام كمجلد محلي

```bash
git clone https://github.com/alwaqad/arcli
cd arcli
```

## الاستخدام

#### كحزمة نظام:
```bash
arcli font1 الوقاد
arcli font2 السلام
arcli font3 تجربة
```

##### مع الألوان:
```bash
arcli --color blue font1 بريمج
arcli -c red font2 رسوميات
arcli --color green font3 عربي
```

##### مع استبدال النقاط:
```bash
arcli --no-dots font1 طرفية
arcli -nd -c green font2 آركلي
```

##### تفاعلي:
```bash
arcli
```

# كملف تنفيذي

لا تنسى تغيير `./arcli` إلى مسار الملف التنفيذي الفعلي في ملفاتك.
```bash
./arcli font1 النص
./arcli --color grey --no-dots font2 النص
```
#### من المجلد المحلي:
```bash
python3 arcli.py font1 النص
python3 arcli.py --color grey --no-dots font2 النص
```

## الخيارات المتاحة

- `font1` - `font6`: اختيار الخط المطلوب
- `--color`, `-c [لون]`: تلوين النص (white, grey, red, green, yellow, blue, pink, magenta, purple, cyan)
- `--no-dots`, `-nd`: استبدال النقاط بمسافات لتحسين المظهر
- `--help`, `-h`: عرض المساعدة

## حذف حزمة النظام

```bash
sudo install-arcli.sh rm
```
